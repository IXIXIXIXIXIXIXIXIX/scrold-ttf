# Scrold

### v1.0.0

## Introduction

Scrold is a monospace font inspired by scrolling digital matrix displays used in railway stations.

## Glyphs

This font was designed primarily as an alphanumeric display font. 

Glyphs have been included for the Latin alphabet alongside various punctuation, but no extended character sets are attempted.

## Installation

Scrold is packaged as a .ttf file and a .woff2 file, so it can be installed, used in an application or on web. Use in an app will vary depending on the technologies that you are using to build the app.

The links below will detail how to install fonts on some of the most popular operating systems:

[Install on Linux](https://wiki.archlinux.org/title/fonts) - I have included a link to the Arch Linux wiki, please see the documentation for your specific distro if you do not use an Arch-based distro.

[Install on Mac](https://support.apple.com/en-gb/HT201749)

[Install on windows](https://support.microsoft.com/en-us/office/add-a-font-b7c5f17c-4426-4b53-967f-455339c564c1)
